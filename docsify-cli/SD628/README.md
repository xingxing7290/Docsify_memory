## 产品功能
+ SD卡存储，支持定时播放
+ 8端口DMX512/TTL输出，支持扩展
+ 连接GPS同步模块，支持无线同步
+ 配合4G物联网模块，支持远程控制。可以编址，调试，调节速度亮度等参数
+ 配合数据线和手机，支持实时控制。可以编址，调试，调节速度亮度等参数
